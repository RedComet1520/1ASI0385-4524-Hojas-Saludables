# Hojas Saludables

## Descripción
Hojas Saludables es una aplicación de solución digital que permite identificar enfermedades en plantas mediante el análisis de imágenes y brindar tratamientos y recomendaciones para su cuidado.

## Segmentos objetivo
- Personas naturales
- Personas agriculas
- Empresas agricultoras

## Características principales
- Escaneo de imágenes.
- Identificación de enfermedades.
- Consejos de cuidado.
- Gestión de plantas registradas.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
